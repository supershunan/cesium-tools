# cesium 工具

## 工具测试

项目启动测试需要将 package.pro.json 文件改为 package.json 进行测试，因为测试的时候是使用react环境进行的测试。
如无特殊说明，工具所有结束操作均为鼠标右键结束操作。

## 发包注意

在发包前一定要检查 package.json 文件的依赖是否是工具包需要的，如果不对，一定是 package.json 和 package.prod.json 搞混了，重新命名即可。版本号需要修改
npm run build
npm publish
